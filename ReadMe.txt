Description:
You work in a toy store that is selling Pokemon. Customers who buy more than one type of Pokemon get a
discount. The same type of Pokemon cannot be added to a discount group.

Files:
Main.java -> Main file of project
Constants.java -> Contains Project constants
Controller.java -> Contains Business Logic
UnitTests.java -> Contains JUnit Test Cases

Extras/Pokemon_Exe.jar -> Executable Jar File (Java -jar Pokemon_Exe.jar)
